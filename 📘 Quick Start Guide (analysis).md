📘 Quick Start Guide (Introduction)
1. Installation

Download the ananeko_gemini_protected.exe.

Prepare your Gemini API Key and YouTube Data API Key.

Launch the exe! (Trial version: 5 launches / 14 days).

2. Database Setup

The app automatically creates a local SQLite database (ananeko.db) on the first launch.

No complex SQL setup is required!

3. Start Analysis

Enter your Gemini API Key in the settings.

Input the YouTube Video ID you want to analyze.

Click "Start Live Analysis" to see real-time viewer sentiment and AI-generated advice.

4. Review Insights

Use the "Refresh Analytics" button to see the ranking of active listeners and comment frequency.
